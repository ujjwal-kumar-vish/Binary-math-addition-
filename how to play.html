3. Game Rules

The player chooses the input via clicking on the given options

The choices are displayed on the 'current binary' screen

After choosing numbers press submit to check

The player gains score if the number chosen is equal to the objective no.

Computer generates random number in a given range

If enough score is earned level changes as well as range of the computer generated random number

The hint is there to help players which displays the answer in binary

Hint costs 2 of the score so the player is not encouraged to use it again and again

The score is there to count the number of successful answer given and the number of hints taken

4. Hint

The hint botton

When pressed the score is deducted by 2

and a binary is displayed which is the currect answer.

This display the final score
